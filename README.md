# A front-end interface for Supar app.
### Quick start

#### Start dev server
```bash
 $ npm run start
```

#### Build the bundle.
```bash
 $ npm run build
```

### NOTES:

- This is not a final product, just me doing some experiments. and learning
more javascript and reactjs.
- This can not work without its backend.
